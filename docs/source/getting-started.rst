Getting Started
===============

Requirements
------------
1. Any Python version >= 2.7

Setup
-----
1. Install.

  .. code-block:: none

    pip install flexdict

2. Import.

  .. code-block:: python

    from flexdict import FlexDict

3. Create.

  .. code-block:: python

    FlexDict()

Usage
-----

Head over to the :ref:`User's Guide` to start using `FlexDict`!
