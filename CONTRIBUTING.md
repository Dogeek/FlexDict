## Contributing
1)  Get the source.
    ```terminal
    git clone https://github.com/ozturkberkay/FlexDict.git
    ```

2)  Install tox.
    ```terminal
    pip install tox
    ```

3)  Test your changes.
    ```terminal
    tox
    ```

4)  If you pass every test, make a PR request using your own branch.
    ```terminal
    git checkout -b mychange
    git push origin mychange
    ```
